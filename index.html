Github is a brilliant code hosting platform, that can be used to save our code projects and effectively tracking them. And also, to get a real world experience we can collaborate our software project with others. But, did you know that?. There is an another fantastic service is provided by Github. That is hosting static website for free, is called Github Pages.

In this blog we will learn about how to host our Static website on github for free by deploying them on github. It's very simple, easy and fast process, within few minutes our website will be live. As a learner or a developer this is a super good service right?!

Note:

There is one limitation. Since it's a static service, we can deploy static websites only. like portfolio, etc (like using HTML, CSS, Javascript). It doesn't support server side languages (like Node.js, Python, PHP, etc.).

to deploy a website on Github

Here is the fast and clear tutorial of hosting a website on github for free. Follow the simple steps to host your website, and share your work with others that you have done. Congrats in advance:)

*To deploy our website on github, first you need a account and repositary on Github.

Creating a new repositary on github

To create a new repositary in github, use + on top right corner on your github page. Then select New Repositary. If you have already one you can use that. For this process creating a seperate repositary is recommended.

Github page --> + --> New Repositary.


URL Name

Once you click the New repositary, it will ask you to give the name for that. That will be part of your website.

Example: https://xxxx.github.io/repositary_name



Add code files

To upload your codes, in the top of github page it shows you Add files option. You can create files and add your code or you can directly upload your codes.

Note:

Here I have uploaded this blog website code folders (HTML, CSS & Images files) by choosing upload option.

Add Files --> Upload Option --> upload your code files.


Pages

After uploading your code files, Go to settings then you will see Pages under general settings.

Settings --> General settings --> Pages.

Branches

In Page settings it shows Branch under the Build and deployment option, then select Mainand just saved it. Also, it shows another option called source. Leave it as default as Deploy from a branch.

Pages --> Build and deployment --> Pages --> Select Main.
Pages --> Build and deployment --> Main --> Deploy from a branch..



Publish

Once you saved your branch setting, within few minutes it will shows yous like your site is live at with your Site URL at the top of your page. Now you can share your link of your site.


Yeah now your website is live! Big Thanks and Wishes!

Still you can change or moderate your codes at anytime eventhough after pubishing it.
Go to file --> Select file which you want to edit --> Select edit --> Edit your code.
